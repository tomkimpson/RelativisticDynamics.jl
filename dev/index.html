RelativisticDynamics.Model — Type
M = BarotropicModel(::Parameters,
                    ::Constants,
                    ::GeoSpectral,
                    ::HorizontalDiffusion)

The BarotropicModel struct holds all other structs that contain precalculated constants, whether scalars or arrays that do not change throughout model integration. In contrast to ShallowWaterModel or PrimitiveEquationModel it does not contain a Boundaries struct as not needed.

RelativisticDynamics.SystemParameters — Type
P = Parameters(kwargs...)

A struct to hold all model parameters that may be changed by the user. The struct uses keywords such that default values can be changed at creation. The default values of the keywords define the default model setup.
